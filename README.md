# Seccamp 2026 mini OAuth 2.0 & OpenID Connect

Test app before hands-on

## How to run locally

### Prerequisite

* node.js 24
* gcloud
* java 26+

### Install

```shell
npm i
```

### Bulid

```shell
npm run build
```

### Run

```shell
npm run emulator & npm run dev
```

Open [http://localhost:8080](http://localhost:8080).
